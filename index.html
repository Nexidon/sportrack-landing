Sportrack

Мобільний застосунок для бакалаврської роботи

Назва роботи

UA: Розробка мобільного застосунку для моніторингу фізичної активності та антропометричних даних "Sportrack".

EN: Development of the "Sportrack" mobile application for monitoring physical activity and anthropometric data.

Мета дослідження

Створення комплексного програмного рішення на базі Android для автоматизації процесу ведення спортивного щоденника, візуалізації прогресу через графіки та ефективного планування тренувального процесу.

Основні завдання

  • Розробка архітектури локального сховища даних на базі бібліотеки Room.
  • Реалізація модулів для фіксації результатів силових та кардіо тренувань.
  • Створення системи візуалізації динаміки ваги та замірів тіла (Measurement Charts).
  • Оптимізація інтерфейсу користувача з використанням сучасних принципів Material Design.

Очікувані результати

Готовий до використання Android-застосунок, який дозволяє користувачу повністю відмовитися від паперових носіїв, забезпечуючи швидкий доступ до історії тренувань та наочний аналіз прогресу.
